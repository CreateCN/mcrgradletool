# MCrGradleTool

#### 介绍
**MCrGradleTool**是由[CreateCN](https://space.bilibili.com/1333235292)开发的基于Go语言编写的命令行工具，  
通过将[该视频](https://www.bilibili.com/video/BV1FeeDepEVK)所提及的方法封装为软件以解决Gradle下载问题

#### 安装教程

从[发行页](https://gitee.com/CreateCN/mcrgradletool/releases/)下载最新版的安装包即可一键安装（当前仅有Windows版）

#### 使用教程

 _注意：本软件为命令行工具，如遇不懂的地方可用`mcrgt --help`获取帮助_ 

1.  先打开MCreator，进行第一次构建，等待构建结束  
ps：如果之前失败过直接运行`mcrgt gradle`即可
2.  构建失败后，运行`mcrgt gradle`，等待软件自动处理
3.  完成！尽情发挥创造力吧！
